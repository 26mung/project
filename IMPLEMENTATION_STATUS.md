## 🚧 진행 중인 작업

### 관리자 페이지 재설계
현재 관리자 페이지를 /app 스타일로 완전히 재구현하는 작업이 필요합니다.

#### 구현 범위 (대규모 작업)
1. **레이아웃 시스템**
   - 사이드바 네비게이션 (280px)
   - 메인 컨텐츠 영역
   - 상단 헤더
   - /app 페이지와 동일한 스타일

2. **대시보드 화면**
   - KPI 카드 (총 사용자, 프로젝트, 관리자, 오늘 가입)
   - 차트 (사용자 추이, 프로젝트 상태)
   - 최근 활동 목록

3. **사용자 관리 화면**
   - 사용자 목록 (검색, 필터, 정렬)
   - 사용자 상세 정보
   - 역할 부여/회수 UI
   - 사용자 삭제 확인 다이얼로그

4. **프로젝트 관리 화면**
   - 전체 프로젝트 목록
   - 프로젝트 상세 정보
   - 생성자 정보 표시

5. **역할 및 권한 화면**
   - 역할 목록
   - 권한 매트릭스
   - 역할별 사용자 수

### 구현 방법 옵션

#### 옵션 1: 단일 HTML 파일 (현재 방식)
- `public/admin.html` 파일 하나에 모든 코드
- 장점: 간단한 배포
- 단점: 유지보수 어려움, 파일 크기 증가

#### 옵션 2: 모듈화된 구조 (권장)
